# 📝 Keeper - Decentralized Notes App  

A Google Keep-inspired note-taking app built on the **Internet Computer (DFINITY)** blockchain using **React + Motoko**.  

![Keeper App Screenshot](/screenshot.png) **(Add a screenshot path)***  

## 🌟 Features  
- **Create/Delete Notes**: Store notes permanently on-chain  
- **Motoko Backend**: Smart contracts handle note operations  
- **React Frontend**: Modern UI with real-time updates  
- **Internet Computer Storage**: Decentralized and tamper-proof  

## 🛠 Tech Stack  
- **Frontend**: React
- **Backend**: Motoko (DFINITY)  
- **Deployment**: Internet Computer Protocol (ICP)
